Comunicación de baja

Giancarlo Ventura Cargando... Léelo en 1 min.

POST /send-summary-document/

Las comunicaciones de baja serán generadas y emitidas a SUNAT con la fecha del día actual.


{
  "fecha_referencia" : "2016-09-08",
  "ruc_emisor": "20538939618",
  "numero": 1,
  "items": [
    {
      "serie": "FF11",
      "numero": "1", 
      "razon_baja": "www.maytok.com",
      "tipo_documento": "01"
    }
  ]
}

Detalle de los parámetros:

Nombre Descripción
fecha_referencia Fecha de generación del documento dado de baja. Esta fecha es diferente a la fecha de generación del documento
numero El número correlativo de la comunicación de baja
items La lista de documentos a los que se desea dar de baja
tipo_documento El tipo de documento de la comunicación de baja. Para más información consultar la tabla resumen
